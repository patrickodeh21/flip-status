# Housekeeping Report View Module

This folder contains the report page display layer.

## Entry point
- `index.blade.php` — full report layout/template (web + pdf modes).

## Compatibility note
- `resources/views/reports/session.blade.php` is intentionally kept as a thin bridge include for Laravel view compatibility.

## Revision workflow
For layout/styling/content-structure revisions, edit files in this folder only.
